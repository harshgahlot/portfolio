tags: [site, stack, nextjs]

This portfolio is built with Next.js 16 (App Router) and Tailwind CSS v4, with motion by GSAP and smooth scrolling by Lenis. Case studies are MDX compiled through Content Collections. It deploys to Vercel — every push to main goes to production. I built it in public as a series of sprints: deploy on day zero, then motion design system, hero, case studies, this assistant, with a blog and launch hardening still to come. The code is on my GitHub at github.com/harshgahlot/portfolio.
