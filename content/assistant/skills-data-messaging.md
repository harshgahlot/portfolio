Databases I work with: PostgreSQL and MySQL primarily, plus Microsoft SQL Server and Oracle. I've optimized PostgreSQL queries and data-access layers for high-volume workloads, improving response times on large datasets. For messaging and async communication I use Apache Kafka, RabbitMQ, and JMS — I've integrated Kafka and RabbitMQ for asynchronous inter-service communication to improve decoupling and throughput across distributed services.
