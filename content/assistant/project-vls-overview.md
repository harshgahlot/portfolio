The Vehicle Location System (VLS) is an enterprise backend I built services for in Java and Spring Boot. Vehicles report GPS positions continuously, and the system ingests that real-time location data at scale, then serves optimized retrieval and analytics APIs on top of it. It's an internal system, so I keep public details at the level I can share — the case study at /work/vehicle-location-system covers the shape of the problem and how I approached it, and I'm happy to go deeper in a conversation.
